Global dashboard for monitoring Corona virus (COVID-19)

## Build & Run

### Prerequisite

```bsh
$ npm install -g nodemon
$ npm install
```

### Development Mode

```bsh
$ npm run build
$ npm start
```

### Production Mode

```bsh
$ npm run build
$ npm run production
```
